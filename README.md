
# 项目结构
* controllers [控制器]
    * database [数据库]
    * tools [工具库]
    * routes [路由库]
    * api [API库]
    * schedule [时间表]
    * index.js [控制器人口]
* public [用来存放静态文件]
* views [视图目录]
* temporary [临时文件目录]
* node_modules [用来存放项目的依赖库]
* app.js [项目入口及程序启动文件]
* package.json [包描述文件及开发者信息]
* README.md [项目说明文件]
